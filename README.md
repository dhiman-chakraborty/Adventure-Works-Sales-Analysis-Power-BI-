# Adventure-Works-Sales-Analysis-Power-BI-
Developed an end-to-end Power BI dashboard for Adventure Works analyzing $24.9M revenue, $10.46M profit, 56K orders, and 18K customers. Included sales trends (YTD/QTD/MOM), product and regional performance, customer segmentation, purchase power, returns, and inventory insights to support data-driven decision-making.
